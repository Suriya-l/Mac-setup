# My Personal Mac Setup

Welcome to my personal macOS configuration repository! This project contains all the dotfiles, configurations, and tools I use to set up my terminal and development environment on a fresh macOS installation.

## What's Included?

- **Zsh**: Custom zsh configuration with aliases, prompts, and syntax highlighting.
- **Neovim**: My personal Neovim editor configuration built on top of NvChad.
- **Modern CLI Tools**: Replaces default terminal tools with modern alternatives (`eza`, `bat`, `fzf`, `zoxide`, `duf`, etc.).
- **Ghostty**: Custom configuration for the Ghostty terminal emulator.
- **GitHub CLI**: Streamlined GitHub integration.

## Installation

To set up a new Mac, simply clone this repository and run the install script:

```bash
git clone https://github.com/Suriya-l/Mac-setup.git
cd Mac-setup/.config_for_mac-main
./install.sh
```

**Note:** Ensure you have [Homebrew](https://brew.sh) installed before running the setup script!
